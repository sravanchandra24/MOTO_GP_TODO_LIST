# MotoGP Leaderboard Web App

A simple web application to add, edit, and delete racing team details with a clean MotoGP-style design.

## Features
- Add Team, Sponsor, Racer, and Wins.
- Edit or delete any entry.
- Validation for empty fields.
- MotoGP-themed UI with dark background and neon accents.
- Responsive table and edit modal.

## Files
- `index.html` – App layout  
- `style.css` – MotoGP racing theme  
- `script.js` – Add/Edit/Delete logic  

## How to Use
1. Open `index.html` in any browser.  
2. Fill the form and click **Submit**.  
3. Use **Edit** or **Delete** to update the table.

## Tech Stack
HTML • CSS • JavaScript • Bootstrap (modal)
